# FLORÉ

### Orthography × Memphis × Illustrated Floral × Impasto

> *Artful systems for brands that bloom.*

FLORÉ is a production-ready design system built at the collision of four aesthetic traditions that have no business being together — and are more powerful for it. Typographic orthography supplies the structural skeleton. Memphis pattern-language supplies the irreverent geometry. Illustrated botanical systems supply the organic warmth. Impasto painting technique supplies the material weight and texture.

The result is a visual language that is simultaneously rigorous and exuberant, systematic and alive.

**Live:** Vercel · **Stack:** Vite · React 18 · TypeScript · Tailwind CSS v3 · Framer Motion · **Org:** [MelodicBloom](https://github.com/MelodicBloom)

---

## The Four Aesthetic Traditions

**Orthography** — the structural layer. Strict typographic hierarchy, grid discipline, measured white space, precise letterspacing. Every layout decision is governed by a legibility and proportion system before any decoration is applied. The bones must be correct.

**Memphis** — the pattern layer. Geometric shapes floating in space — squiggles, dots, triangles, bold stripes. Memphis refuses to be serious. It insists that decoration is not frivolous but essential. In FLORÉ, Memphis geometry is generated by `MemphisDecor.tsx`, a procedural SVG system that places ornament without repeating it.

**Illustrated Floral** — the warmth layer. Botanical forms — petals, stems, leaves, seed pods — drawn with illustrator intent rather than icon simplicity. These are not clipart. They carry the imprecision of the human hand. In FLORÉ, botanical forms are used as section markers, dividers, and hover-state flourishes.

**Impasto** — the texture layer. Thick paint, visible brushwork, dimensional surface quality. Applied via CSS `box-shadow` stacking, SVG feTurbulence, and carefully composed gradient layering that simulates paint-body on screen. Nothing in FLORÉ is flat.

## Design Tokens

### Color Palette

| Token | Hex | Role |
|-------|-----|------|
| Ink | `#0E0F14` | Primary text, deepest shadow |
| Ivory | `#F7F3EE` | Base surface, unprinted warmth |
| Blush | `#F2D7D9` | Soft botanical accent, hover wash |
| Sage | `#C7D8C0` | Botanical green, growth emphasis |
| Lavender | `#8D7CC3` | Memphis geometry, primary accent |
| Ochre | `#E9C46A` | Impasto warm highlight, Memphis spot |
| Deep Green | `#123B2D` | Dark botanical, grounding shadow |
| Charcoal | `#222222` | Body text, secondary surfaces |

### Typography

| Role | Family | Weight | Usage |
|------|--------|--------|-------|
| Display | Playfair Display | 700–900 | Section headlines, orthographic authority |
| Subheading | DM Serif Display | 400 | Botanical section labels, editorial entries |
| Body | DM Sans | 400–500 | Long-form reading, clean utility |
| Label | Space Mono | 400 | Metadata, token values, Memphis annotation |

### Motion

FLORÉ uses Framer Motion with two primary easing signatures:
- **Botanical unfurl** — `spring({ stiffness: 60, damping: 15 })` — organic, slightly oscillating entrance
- **Memphis snap** — `easeOut` at 0.25s — crisp, geometric, deliberate

All motion respects `prefers-reduced-motion`.

```

    App.tsx                    ← Root layout, section orchestration
  components/
    layout/
      Header.tsx               ← Orthographic nav — strict grid, Memphis accent strip
      Footer.tsx               ← Botanical colophon footer
    sections/
      Hero.tsx                 ← Impasto hero — layered paint-texture backgrounds

      SelectedWork.tsx         ← Portfolio cards with botanical hover flourishes
      About.tsx                ← Studio story with illustrated floral dividers
      Journal.tsx              ← Editorial blog entries, orthographic column layout
      CTA.tsx                  ← Memphis-pattern call-to-action panel
    ui/
      Button.tsx               ← 3 variants: primary (impasto), ghost (orthographic), Memphis
      Card.tsx                 ← Botanical card with paint-layer shadow system
      ColorSwatch.tsx          ← Living palette specimen with token labels
      MemphisDecor.tsx         ← Procedural SVG Memphis geometry generator
      PaintingPanel.tsx        ← Impasto-texture surface panel component
      TokenTable.tsx           ← Design token display table
      IconRow.tsx              ← Botanical icon row with Lucide + custom flourishes
    motion/
      Reveal.tsx               ← Botanical unfurl scroll-reveal wrapper
  data/
    tokens.ts                  ← Color, type, spacing token registry
    projects.ts                ← Portfolio project metadata
    journal.ts                 ← Journal entry content
  styles/
    globals.css                ← CSS custom properties, impasto filter system
```

## Why These Four Aesthetics Together

Orthography, Memphis, botanical illustration, and impasto painting are not arbitrary choices. Each one is a different theory of what design is *for*:

- Orthography says design is *communication* — precision, hierarchy, legibility.
- Memphis says design is *joy* — pattern, play, the refusal of solemnity.
- Botanical illustration says design is *observation* — looking closely, drawing what is actually there.
- Impasto says design is *making* — the material presence of the hand, the texture of effort.

FLORÉ holds all four simultaneously. A brand using this system communicates precisely, plays freely, observes carefully, and makes visibly. That is the intended tonal identity: **disciplined exuberance**.
